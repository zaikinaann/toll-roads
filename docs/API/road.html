Получения деталей маршрута между точками от API Яндекса

Authorizations:
BearerAuth
query Parameters
apikey
required
string
Example: apikey=my_hacked_access_key

Ключ, полученный в Кабинете Разработчика

waypoints
required
string
Example: waypoints=25.234369457896325,55.280222457968712|25.234369457896325,55.401544758961258

Точки маршрутов, указанные в десятичных градусах (стандарт WGS84)

mode
string
Enum: "driving" "truck"

Режим маршрута

departure_time
integer

Время отправления в формате Unix

avoid_tolls
boolean

Избегать платных дорог (по умолчанию false)

avoid_zones
Array of strings
Example: avoid_zones=lat1,lon1&avoid_zones=lat2,lon2

Избегать зоны в формате lat,lon

traffic
string
Example: traffic=disabled

Учет трафика

weight
number <float>
Example: weight=1500

Масса ТС в тоннах

axle_weight
number <float>
Example: axle_weight=500

Фактическая нагрузка на ось ТС в тоннах

max_weight
number <float>
Example: max_weight=3000

Разрешенная максимальная масса ТС в тоннах

height
number <float>
Example: height=4

Высота ТС в метрах

width
number <float>
Example: width=2

Ширина ТС в метрах

length
number <float>
Example: length=8

Длина ТС в метрах

payload
number <float>
Example: payload=1000

Максимальная грузоподъёмность ТС в тоннах

eco_class
integer
Example: eco_class=3

Экологический класс ТС

has_trailer
boolean
Example: has_trailer=true
